# Executive Sales Dashboard

## Project Summary

Built an executive dashboard in Tableau to analyze revenue and profit performance.

## Business Questions

- Which region generates the most revenue?
- Which product generates the most profit?
- Which sales representative generates the most revenue?

## Dashboard Components

- Revenue by Region
- Profit by Product
- Revenue by Sales Rep

## Tools Used

- Tableau
- Excel

## Key Findings

- North region generated the highest revenue.
- Laptop sales generated the highest profit.
- Sales Rep R001 generated the highest revenue.
