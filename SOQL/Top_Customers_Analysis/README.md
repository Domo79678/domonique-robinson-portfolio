# Top Customers Analysis (SOQL)

## Business Problem

Leadership wants to identify the highest value customers in Salesforce to prioritize account management efforts and improve customer retention.

## Goal

Use SOQL queries to identify:

- Top customers by revenue
- Largest opportunities
- Recently closed deals
- Pipeline value
- Open opportunities
- Account activity

## Tools Used

- Salesforce
- SOQL
- VS Code

## Skills Demonstrated

- Data retrieval
- Business analysis
- CRM reporting
- Query writing
- Sales analytics
