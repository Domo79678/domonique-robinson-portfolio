# JobRadar

## Problem

Job seekers receive dozens of job alert emails every day from LinkedIn, Indeed, CareerBuilder, Google Alerts, and other platforms.

Reviewing every email manually is time-consuming and causes qualified opportunities to be missed.

## Solution

JobRadar automatically scans job alert emails, extracts job opportunities, evaluates them against user-defined criteria, and produces a prioritized list of opportunities.

## Target User

Job seekers receiving large volumes of job alerts.

## MVP Features

* Gmail Integration
* Job Alert Extraction
* Job Fit Scoring
* Apply / Review / Skip Classification
* CSV Export

## Success Metrics

* Reduce manual review time
* Surface high-fit opportunities first
* Increase application efficiency
* Reduce inbox overload
