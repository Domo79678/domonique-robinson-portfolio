# Portfolio Auditor

## Overview

Portfolio Auditor is a Python utility that scans my technical portfolio for missing documentation and project assets.

## Features

- Checks for README.md files
- Checks for screenshots
- Checks for project files
- Calculates a documentation score
- Generates recommendations
- Exports an audit report

## Technologies

- Python
- pathlib
- datetime

## Future Improvements

- Recursive scanning
- HTML reports
- CSV export
- Automatic GitHub statistics
